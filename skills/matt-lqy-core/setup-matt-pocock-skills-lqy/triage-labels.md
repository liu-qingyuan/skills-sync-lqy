# Triage 标签

这些 skills 使用五个规范 triage role。此文件把这些 role 映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills role | 本仓库 tracker 中的标签 | 含义 |
| ---------------------- | ---------------------- | ---- |
| `needs-triage` | `needs-triage` | 维护者需要评估该 issue |
| `needs-info` | `needs-info` | 等待报告者提供更多信息 |
| `ready-for-agent` | `ready-for-agent` | 已完全说明，AFK agent 可以接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实施 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个 role 时（例如“应用 `ready-for-agent` 标签”），请使用此表中对应的标签字符串。

编辑右侧列，让它匹配你实际使用的标签词汇。
