# 分类标签

这些技能以五个典型的分类角色来表示。该文件将这些角色映射到该存储库的问题跟踪器中使用的实际标签字符串。

| mattpocock/技能标签 |我们的追踪器中的标签 |意义|
| -------------------------- | -------------------- | ---------------------------------------------------- |
| `需求分类` | `需求分类` |维护者需要评估这个问题|
| `需求信息` | `需求信息` |等待记者进一步了解|
| `准备好代理` | `准备好代理` |完全指定，为 AFK 特工做好准备 |
| “为人类做好准备” | “为人类做好准备” |需要人工实施 |
| `不会修复` | `不会修复` |不会采取行动|

当技能提到角色时（例如“应用 AFK 就绪分类标签”），请使用此表中相应的标签字符串。
